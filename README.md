# 📊 Análise Exploratória de Dados de Vendas com Python

## 📌 Sobre o Projeto
Este projeto apresenta uma análise exploratória de dados de vendas utilizando Python.  
O objetivo é identificar padrões de receita, desempenho de produtos e comportamento das vendas ao longo do tempo.

## 🎯 Objetivos
- Analisar o faturamento total
- Identificar os produtos mais vendidos
- Avaliar o desempenho das vendas ao longo do tempo
- Calcular o ticket médio
- Gerar insights a partir dos dados

## 🛠 Tecnologias Utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Estrutura do Projeto

data/ → arquivos de dados utilizados na análise  
notebooks/ → notebook com a análise exploratória  
src/ → scripts Python do projeto  

## 🚀 Como Executar
1. Clone o repositório
2. Instale as dependências:

pip install -r requirements.txt

3. Execute o notebook localizado em:

notebooks/analise.ipynb
