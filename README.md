# 📊 Análise Exploratória de Dados de Vendas com Python

## 📌 Objetivo
Este projeto tem como objetivo realizar uma análise exploratória de dados (EDA) em um conjunto de dados de vendas para identificar padrões, tendências e insights que possam apoiar a tomada de decisão.

## 🧰 Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Estrutura do Projeto

analise-exploratoria-vendas-python
│
├── dados/ → dataset utilizado na análise
├── cadernos/ → notebooks de análise exploratória
├── src/ → scripts Python
├── README.md → documentação do projeto
└── requisitos.txt → bibliotecas utilizadas

## 📊 Etapas da análise
- Importação e exploração inicial dos dados
- Limpeza e tratamento de dados
- Análise estatística básica
- Visualização de dados
- Identificação de padrões de vendas

## 📈 Exemplos de análises
- Produtos mais vendidos
- Distribuição de vendas por categoria
- Tendências de vendas ao longo do tempo
- Comparação de desempenho entre produtos

## 🚀 Como executar o projeto

1. Clone o repositório

```bash
git clone https://github.com/Thamixalvez/analise-exploratoria-vendas-python.git
```

2. Instale as dependências

```bash
pip install -r requisitos.txt
```

3. Abra o notebook na pasta `cadernos`
